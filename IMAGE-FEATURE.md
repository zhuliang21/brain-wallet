# 📸 极简图片生成功能

## 功能概述
Brain Wallet 现在支持自动生成极简风格的钱包图片，包含QR码和助记词信息。

## 图片特性
- **尺寸**: 800x400像素（高清晰度）
- **布局**: 左右两个等大区域
- **风格**: 极简纯净设计，无边框分割线
- **字体**: 等宽字体（monospace）
- **内容**: 核心信息，无多余文字
- **质量**: 高DPI渲染，文字清晰
- **交互**: 点击图片即可下载

## 图片内容
### 左侧区域（400x400）
- QR码（320x320像素）
- 包含完整助记词信息
- 可直接扫描导入钱包
- 居中显示

### 右侧区域（400x400）
#### 上半部分（400x200）
- 输入文本（前60个字符）
- 16px等宽字体
- 水平和垂直完全居中

#### 下半部分（400x200）
- 12个助记词整体居中
- 格式: "1.word 2.word 3.word"
- 3列4行网格排列
- 14px等宽字体
- 整个助记词块在区域内居中

## 使用方法
1. 输入文本生成钱包
2. 图片自动显示在页面上
3. 点击图片即可下载PNG文件
4. 悬停图片有放大效果

## 技术特点
- Canvas API高质量渲染
- 高DPI自适应（支持Retina显示）
- 自动生成，无需手动操作
- 点击下载功能
- 悬停交互效果
- 支持中英文界面
- 响应式设计

## 布局设计
- 左右等分：各400px宽
- 右侧上下等分：各200px高
- 无边框线，纯净设计
- 所有内容完全居中对齐
- 点击图片下载提示
